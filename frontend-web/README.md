#  Enterprise Retail Strategic Inventory System (Frontend)

>  Currently under development — Cashier module in progress

A modern **React + Vite** based frontend for a smart retail management system.  
This project currently includes the **Cashier Dashboard** and is designed to scale with additional modules like Admin, Inventory, and Analytics.

---

##  Features (Current - Cashier)

-  Billing Interface  
- Product Listing  
-  Add / Remove Items  
-  Total Calculation  
-  Fast and responsive UI (Vite + React)  
-  Reusable components structure
